10 recursos gratuitos para desarrollo web

Plataforma de aprendizaje:

Khan Academy
Está disponible en el dominio https://es.khanacademy.org y es una plataforma online de aprendizaje para aprender muchos conocimientos variados (matemáticas, química, biología...). Para aprender programación hay cursos principalmente de HTML, CSS, JavaScript y SQL.
Incluye lecciones, ejercicios y evaluaciones para ayudar al proceso de aprendizaje. Está disponible en varios idiomas.


Plataforma independiente:

6Nomads Esta web está disponible en el dominio https://6nomads.com.
Es una plataforma que pone en contacto a desarrolladores, ingenieros, etc con empresas que necesitan estos trabajadores tecnológicos en remoto. Hay que registrarse, tanto si eres empresa como si eres trabajador, se crea un perfil y sólo es cuestión de que ambas partes se pongan en contacto.


Ilustraciones:

Drawkit Esta web está disponible en https://www.drawkit.com. Es una web para comprar ilustraciones y recursos gráficos para distintos usos. Los recursos están disponibles en varios formatos (svg, png...).
Se puede comprar una suscripción o acceder únicamente a los recursos gratuitos que ofrece la página.
Los distintos autores pueden registrarse y subir su contenido, crear distintas colecciones, etc.


Iconos:

Font Awesome Es una librería de iconos vectoriales muy útil y muy fácil de instalar. Hay una colección básica de iconos pero es ampliable y se pueden incluso personalizar.
La biblioteca completa y cómo instalarlo se puede consultar en la web https://fontawesome.com.
Sin ninguna duda una de las biblliotecas más interesantes que se usan en la web.


Tipografías:

Font Squirrel Una web con recursos tipográficos que se puede consultar en la siguiente URL:https://www.fontsquirrel.com.
Permite descargar y utilizar de forma libre muchas tipografías. Hay que consultar la licencia porque no todas son gratis o no todas tienen un uso personal.
De todas formas también es útil que se puedan comprar bajo licencia algunas tipografías.
